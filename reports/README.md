# Reports

Auto-generated reports from database.

## Reports

| Report | Output | Audience |
|--------|--------|----------|
| QuickLook CSV | CSV file in Google Drive | Leadership |
| Quad Chart Data | JSON for slide generation | Weekly meetings |
| Status Reports | PDF/HTML summary | Team |

## Files

| File | Purpose |
|------|---------|
| `quicklook-generator.gs` | Generate milestone CSV |
| `quadchart-data.gs` | Format data for quad chart slides |

## QuickLook CSV

Columns:
- Solution
- Phase
- Solution Kickoff
- ATP DG Memo
- F2I DG Memo
- ORR Memo
- Production Start
- Closeout

## Quad Chart Data

Sections:
- Solutions Updated This Week
- Upcoming Milestones
- Open Action Items
- Key Decisions Needed
