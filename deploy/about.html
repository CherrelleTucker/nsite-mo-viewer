Platform Overview

MO-Viewer is a unified dashboard for the NSITE Management Office that consolidates solution tracking, stakeholder engagement, and action management into a single interface.

Security & Access

Contains PII — This application displays stakeholder contact information including names, emails, phone numbers, and organizational affiliations.

Hosting Environment

MO-Viewer is a Google Apps Script web application hosted within the OSDI Google Workspace environment. All data remains within Google's infrastructure and is subject to NASA OSDI security policies.

Access Control

Access is restricted to an approved whitelist of users. To gain access:

  1. Submit an access request to the MO-Viewer administrator
  2. Upon approval, your Google account email is added to the whitelist
  3. Access the application using your approved Google account
Authentication Google OAuth 2.0 — users must sign in with their Google account
Authorization Email whitelist stored in Script Properties — only approved emails can access
Data Storage Google Sheets within OSDI Workspace — no external data transfer

Request Access

Contact the MO-Viewer administrator to request access. Include your Google account email and justification for access.
